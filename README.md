# Zero Trust Cybersecurity Governance Project

Developed a Zero Trust-focused cybersecurity governance and compliance framework for a cloud-based accounting firm aligned with NIST CSF 2.0 principles.

## Technologies & Frameworks
- Zero Trust Architecture (ZTA)
- NIST CSF 2.0
- Microsoft Entra ID
- Microsoft Sentinel
- Microsoft Defender
- Active Directory
- Kali Linux
- Wireshark
- pfSense

## Project Objectives
- Strengthen identity and access management
- Implement MFA and RBAC strategies
- Improve cloud governance and compliance
- Reduce insider threat and credential compromise risks
- Develop incident response and monitoring policies

## My Contributions
- Conducted cybersecurity risk analysis
- Developed Zero Trust governance policies
- Designed compliance and monitoring strategies
- Mapped controls to NIST CSF 2.0
- Assisted with risk management and incident response planning

## Outcome
The project proposed a governance-driven cybersecurity framework to improve organizational security posture and advance the organization from NIST CSF Tier 1 to Tier 2 (Risk-Informed).
